This repository includes the incidence matrices of Bach's solo violin compositions. Connes's distance, node and edge curvatures, and spectral dimension can all be computed using them. 

The simplicial complexes construction follows this paper:
Mrad, Dima, and Najem, Sara "Higher-order network representation of JS Bach’s Solo Violin Sonatas and Partitas: Topological and geometrical explorations." Chaos, Solitons & Fractals 207 (2026): 117993, with the illustration given below. 
![Demo](InteractiveTwinkleStarEvolution.gif)

The distance and curvature computation can be computed following this paper:

Najem, S., Mrad, Dima., M. Elsayed "Geometric Features of Higher-Order Networks via the Spectral Triplet". Nature Communication Physics (2026).
